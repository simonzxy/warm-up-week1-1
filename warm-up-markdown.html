试题说明

之后热身练习与考核测试都以此形式进行

  • 收到考核邮件请回复邮件以确认，做完测试题后请再回复邮件表示交卷（记得放上 github 项目链接）。
  • 每道试题做完后请提交信息至github账号本次项目下。
  • 顺序做题。
  • 考核时间: 周日（16/06/12）12:00 前交卷 。
  • 有任何疑惑请在微信里及时交流。

git 提交约束

  • 提交信息以增加（add）、优化（optimized）、纠正（fixed)开头。（冒号空格，然后提交信息）
  • git commit 提交内容,请勿擅写。
  • 允许重复提交。（例如提交第一题后，发现有 bug 或更好的实现方式，可以提交纠正或优化）
  • 所有 git 操作请使用命令行。
